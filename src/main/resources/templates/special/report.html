현재 페이지는 화면 테스트용 리포트입니다.

1. 계정 권한 상태

현재 특별계정은 SPECIAL 권한으로 보호된 페이지에 접근할 수 있습니다.

2. 접근 제한 상태

`/special/**` 경로는 SPECIAL 권한을 가진 계정만 접근할 수 있도록 설정되어 있습니다.

3. 내부 도구 상태

내부 도구 페이지, 실행 로그 페이지, 리포트 페이지가 정상적으로 연결되었습니다.

아직 새 리포트가 생성되지 않았습니다.
